# MSL本体配置

这是MSL的设置页面:

![](./assets/msl_settings.png)

功能调整中基础功能的三个按钮不再一一介绍

在```语言```中可以更改语言

在```添加下载任务```中可以将链接复制到这里，点击确定即可下载

```自动化选项```可以使你的体验更加便捷，但是自动运行Frpc需要你使用Frp的标识码

E.g.:我的一个Frp标识码是12345678，我需要让他开机启动，则我需要在方框内填写`12345678,`然后开启功能
